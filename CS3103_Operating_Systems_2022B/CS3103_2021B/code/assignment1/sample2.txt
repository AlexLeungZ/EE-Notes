
//Basic instructions for mini-shell:
Tab Key:        Files suggestion
Arrow Key Up:   Previous arguments
Arrow Key Down: Next arguments

//Supported Commands (basic):
fg: Run process in background
bg: Run process in foreground
list: List all running or stopped processes
ls: Same as "list"
exit: Exit program and terminate all running processes
ex: Same as "exit"

//Supported Commands (supplementary):
clear: Clear terminal screen
cls: Same as "clear"
date: Check current date and time
g++ver: Check the version of g++ installed

//*Command demos//

//Run the code
$ g++ 56046680.cpp -lreadline -lpthread -o 56046680 && ./56046680
Mini-Shell build by c++
sh >

//Type fg
sh >fg ./test hello 2 5
...(output from test)...
sh >

//Type bg
sh >bg ./test hello 2 5
sh >...(output from test)...
(input here)

//list or ls (when no running or stopped processes)
sh >list
sh >

//exit or ex (when no running process)
